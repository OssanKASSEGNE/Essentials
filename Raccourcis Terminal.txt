Raccourcis Terminal Linux


Liste : 
espace = page suivante
entrée = ligne suivante
q = quitter

ctrl +A : début
ctrl +E : fin
ctrl +U : effacer ligne
ctrl + W : effacer mot
ctrl Y : coller ce qui vient d’être effacé.
ctrl + D quitter root
