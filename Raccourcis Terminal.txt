Raccourcis Terminal Linux


Liste : 
espace = page suivante
entrée = ligne suivante
q = quitter


ctrl +A : début
ctrl +E : fin
ctrl +U : efface Tous ce qui est à GAUCHE du curseur
ctrl +K : Supprime tout ce qui est à droite du curseur
ctrl + W : effacer mot
ctrl Y : coller ce qui vient d’être effacé.
ctrl + D  quitter le terminal ou root
	: exit

INFORMATIONS SUR LE DISQUE

du : taille sur le disque
        -h taille en octet
        -a all
        -s regroupée

RECHERCHE DE COMMANDES

history  : retouver historique des commandes
Ctrl + R : cherche une commande

DEPLACEMENT DANS LE TERMINAL

shift + pgUp	: Monter
